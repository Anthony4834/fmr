# fmr.fyi - Fair Market Rent Data Platform

fmr.fyi provides instant access to Fair Market Rent (FMR) data from the U.S. Department of Housing and Urban Development (HUD). Our platform makes it easy to search, explore, and analyze FMR data to help you understand rental market trends and make informed real estate decisions.

## Search & Explore

**Multiple Search Options**
- Search by **ZIP code** for precise, location-specific data
- Search by **city and state** to see city-wide FMR values
- Search by **county and state** for county-level data
- Search by **address** with automatic geocoding to find the nearest FMR data
- Browse popular searches, top markets, and trending locations

**Comprehensive FMR Data**
- View Fair Market Rent (FMR) and Small Area FMR (SAFMR) data for 0-4+ bedroom units
- See year-over-year (YoY) growth percentages
- View 3-year compound annual growth rate (CAGR)
- Access historical FMR data from FY 2022 through FY 2026
- Compare ZIP codes within counties and cities
- Identify which areas use SAFMR (ZIP-level) vs standard FMR (county/metro-level)

## Investment Analysis

**Investment Scores**
- Get investment scores for ZIP codes, cities, and counties
- See net yield calculations based on property values and rent
- View rent-to-price ratios to identify value opportunities
- Compare investment potential across different locations

**Market Insights**
- View ZIP code rankings within counties and cities
- See how individual ZIPs compare to county and city medians
- Access property value data (Zillow Home Value Index integration)
- View property tax rates and mortgage rates by ZIP code
- Analyze market parameters for informed decision-making

**Historical Trends**
- Interactive charts showing FMR trends over time
- Compare multiple years of data side-by-side
- Identify growth patterns and market stability
- View bedroom scaling curves to understand pricing efficiency

## Browse & Discover

**Browse Pages**
- Browse all cities with FMR data
- Explore counties across all states
- Discover ZIP codes with detailed breakdowns
- Filter and sort by various metrics

**Dashboard & Statistics**
- Nationwide statistics and insights
- State-level dashboards with comprehensive metrics
- Popular searches and trending locations
- Top markets by average FMR
- Price jump analysis showing significant changes

## Data Sources

- **Fair Market Rent (FMR)**: Official HUD data updated annually
- **Small Area FMR (SAFMR)**: ZIP code-level FMR data for select metropolitan areas
- **Property Values**: Zillow Home Value Index (ZHVI) data
- **Market Parameters**: Property tax rates and mortgage rates by ZIP code

## Key Features

- **Fast, Accurate Search** - Find FMR data instantly with our powerful search
- **Historical Data** - Access multiple years of FMR data for trend analysis
