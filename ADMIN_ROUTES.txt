FULL LIST OF UNLISTED ADMIN/ANALYTICS VIEWS
============================================

1. TEST COVERAGE DASHBOARD
   URL: /test-coverage
   Purpose: Data coverage analytics and quality monitoring
   Features:
     - Summary statistics (cities, ZIPs, counties with/without FMR)
     - Multiple tabs: Cities, ZIPs, Counties, ZIP Mappings, Missing Mappings, Invalid State Codes
     - Filterable by state, year, issue type
     - Export functionality for detailed reports
     - Shows SAFMR vs FMR coverage gaps
     - Tracks mapping issues and data completeness

2. MISSING DATA EVENTS DEBUG PAGE
   URL: /debug/missing-data
   Purpose: Track missing data events from cash flow calculations
   Features:
     - Summary cards showing total events, unique issues, unique ZIPs, sources
     - Breakdown by missing field type (tax rate, mortgage rate, FMR data, etc.)
     - Filterable table with pagination
     - Filters: ZIP code, missing field, source (chrome-extension)
     - Shows occurrence counts and timestamps
     - Helps identify data gaps in the system

3. INCOMPLETE GEOGRAPHIES PAGE
   URL: /incomplete-geos
   Purpose: List locations with investment scores but missing full FMR data
   Features:
     - Filter by type (city/county) and state
     - Shows median investment score and ZIP count
     - Links to location detail pages
     - Explains why locations are incomplete
     - Helps identify areas needing better data mapping

4. MARKET EXPLORER (Public but not in sitemap)
   URL: /explorer
   Purpose: Browse and search markets by investment score
   Status: Has SEO metadata but not included in sitemap.xml
   Note: May be intentionally unlisted or oversight

5. MARKET INSIGHTS (Public but not in sitemap)
   URL: /insights
   Purpose: Market intelligence dashboard (trends, movers, anomalies)
   Status: Has SEO metadata but not included in sitemap.xml
   Note: May be intentionally unlisted or oversight

API ROUTES (Admin/Debug)
=========================
- /api/test-coverage - Test coverage data API
- /api/debug/missing-data-events - Missing data events API
- /api/stats/incomplete-geos - Incomplete geographies API

NOTES
=====
- All admin pages are currently publicly accessible (no authentication required)
- None of these routes are included in sitemap.xml
- None are blocked by robots.txt (only /api/ is disallowed)
- These are internal tools for monitoring data quality and coverage
- Consider adding authentication or IP restrictions for production use
